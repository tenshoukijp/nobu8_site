Modのダウンロード

動作環境

  • Windows 10 以上
    (管理人の動作確認は Windows 11 のみ)
  • .NET Framework 4.8以上
    (わざわざWindowsの更新を強制的に止め続けているとかでない限り、.NET Framework 4.8.1が入っているハズです)

ダウンロード

更新日 %(year)04d/%(mon)02d/%(mday)02d
烈風伝.mod
ダウンロードした後、解凍する前に、zipを「ブロック解除(or 許可する)」必須

zipを解凍する際には、「プロパティ」から「ブロック解除(or 許可する)」してください。
「プロパティ」に「ブロック解除(or 許可する)」といった項目が無いようならこの作業は不要です。

烈風伝ゲームのフォルダーへファイル群をコピー

ゲーム本体のファイルが上書きされたりはしません。
(ゲーム本体のファイル名とは被っていない)

Microsoft VC++ 再頒布可能パッケージをインストール

Visual Studio 2022 C++ ランタイム
x86版をインストールしたことが無い方はインストールしてください。
お手持ちのOSが64bitか32bitかには関わりなく、x86版のインストールが必要です。

以上で終了です。
